Docker
Compose



Docker Compose es una herramienta que simplifica el proceso de ejecutar múltiples contenedores Docker como una sola aplicación. Con Docker Compose, puedes definir y configurar todos los servicios requeridos por tu aplicación en un solo archivo, y fácilmente iniciar y detener todos los contenedores que componen tu aplicación.

Docker compose file

Ejemplo de un archivo docker-compose.yml

                    
                      
                      version: "3.9"
services:
  web:
    build: .
    ports:
      - "8000:5000"
  redis:
    image: "redis:alpine"

Levantar y bajar servicios

Levantar un servicio

                    
                      docker compose up

Bajar un servicio

                    
                      docker compose down
