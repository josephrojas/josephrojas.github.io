Imágenes

Esta será la base de nuestros contenedores, basándonos de un Dockerfile se monta la imagen, la cual será la base para nuestro contenedor

DockerFile

Conjunto de instrucciones que definirán la imagen de nuestro contenedor

Ejemplo Dockefile

                    
                      
                      FROM node:18-alpine
                      WORKDIR /app
                      COPY . .
                      RUN yarn install --production
                      CMD ["node", "src/index.js"]
                      EXPOSE 3000

Estructura de un Dockerfile

Un Dockerfile puede implementar algunas de las siguientes operaciones

FROM

De cual imagen base vamos a partir para la creación de nuestra nueva imagen

COPY

Copiaremos de un contexto a la ruta de nuestro contenedor

RUN

Ejecuta una acción cualquiera

Capas de un contenedor

Para optimizar la creación de un contenedor y el uso de este, las imágenes manejan una estructura de capas, donde cada acción es una capa nueva. Uso de layer cache

Ejecución de un dockerfile 

                    
                      docker build -t <nombre> <contexto_build>
